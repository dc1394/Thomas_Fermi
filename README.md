★これは何？
　原子に対して、Thomas-Fermi方程式を有限要素法（Finite Element Method、FEM）で、
　数値的に解くプログラムです。
　ビルドには、以下のライブラリが必要です。
　・Boost C++ Libraries
　・Eigen
　・GNU Scientific Library
